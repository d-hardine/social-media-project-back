# social-media-project-back

The backend of the Hardine Book app

- Demo for Hardine Book: [Live](https://hardine-book.netlify.app)
- Frontend for [Hardine Book repo](https://github.com/d-hardine/hardine-book-front)

## Built with
- ExpressJS
- Postgresql

## Tools 
- **Cloudinary** : cloud based media management service 
- **Prisma**: ORM to connect and query database
- **Socket.IO**: for live communication between client and server
